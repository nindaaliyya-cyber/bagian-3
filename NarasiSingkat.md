Yang saya ketahui terkait Naming Convention adalah konsep dimana ingin menamai suatu kode agar lebih jelas dan mudah untuk dipahami, selain itu dengan menggunakan ini maka dapat mempemudah kita untuk membedakan suatu kode. Penggunaan Naming Convention juga tidak diasrankan hanya dengan menggunakan satu atau dua huruf maupun angka.

Penamaan variabel dan class sangat penting, penamaan variabel penting agar dapat membedakan variabel atu dengan yang lainnya agar saat di run kode dapat dengan mudah untuk dibaca oleh sistem sedangkan penamaan class sangat penting karena agar lebih terstruktur.

Operator aritmatika dalam java bekerja dengan beberapa hal misalnya perhitungan matematis yaitu penjumlahan, pengurangan, perkalian dan pembagian. Java juga bisa membaca beberapa operator yang digabungkan. Dan urutan operatornya dilakukan sesuai dengan kaidah matematika yang biasa kita ketahui, atau juga membaca dari awal.

Penerapan yang saya pahami dari latihan sebelumnya adalah bagaimana cara mengubah nilai secara bertahap, menghitung beberapa hal baik itu berupa harga, umur, banyak suatu benda dan lain lain.

Kesulitan yang saya alami dalam memahami materi ini adalah terkadang bingung saat harus menambahkan suatu variabel saat menggunakan suatu tipe data yang berbeda. Dan walapun terkadang lebih ke kurang telilti karena terkadang kodenya memiliki beberapa kata typo yang seharusnya tidak ada dikode tersebut ataupun kurang suatu variabel atau keslahan penulisan, tapi setelah megecek kembali akhirnya saya bisa memperbaiki kode tersebut agar tidak error.
